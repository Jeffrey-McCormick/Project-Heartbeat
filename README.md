Hello! Welcome to Project Heartbeat, my individual project for my ENGR 355 - Embedded System Design class.
This project's first revision is finished and turned in for credit. Later versions may appear as I have opportunity to expand this heartrate monitor's capabilities. I used KiCad and ARM Keil applications for designing and programming the board. 

I encourage you to read the Project Heartbeat Design Report to understand my design approach as well as what the functions of my code are for. 
Although, you should be able to read the comments in my code and understand what is happening.
